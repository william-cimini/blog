Skip waiting for internet domain name server configuration changes

by William Cimini

Cause

I was recently testing different name server providers for my internet domain, but I did not have a lot of time to wait for DNS to propagate.

Potential solution

Google provides a Flush Cache tool which will refresh Google’s Public DNS cache.

While the Flush Cache tool is not guaranteed to work for every scenario, I was able to utilize the tool multiple times in a row to skip waiting hours for the latest configured name servers of my internet domain to resolve.

Your computer or network service most likely has to have Google Public DNS configured as the name server, or make use of another DNS service which operates closely to Google’s.
