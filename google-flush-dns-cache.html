Change name servers for your internet domain? Pontentially skip the wait with Google’s Flush Cache tool

Changing name server providers for your internet domain typically takes a while to propagate on the internet.

Google provides a https://developers.google.com/speed/public-dns/cache tool which will refresh Google’s Public DNS cache.

While not gauranteed to work, using this tool on my domain caused the latest name servers to resolve on my computer.
